# expression_template

This project is intended to test the performance difference 
between creating DAG with 
- handwritten every expression
- Use virtual inheritence
- Use expression template 

## tools to use 

- gcc
- clang
- clang-format
- clang-tidy
- cppcheck

- CMake
- conan

- google benchmark
